### 1.单元测试小试牛刀

`MyClass.DoubleValue();`

```c#
[TestMethod]
public void DoubleValueTest()
{
//1. Arrange:准备
MyClass target = new MyClass();
//2. 设定测试用例
int value = 1;
int expected = 2;
//3. Act:执行
int actual = target.DoubleValue(value);
//4. Assert:断言
Assert.AreEqual(expected, actual);
}
```
