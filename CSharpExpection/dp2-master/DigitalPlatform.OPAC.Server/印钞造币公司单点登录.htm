单点认证Webservice说明

 

http://portal.cbpmc.cbpm/AuthCenter/services/validate?wsdl

（具体内容参考validate.xml）

 

认证接口说明

方法定义

validate(string ssoTokenId, String appId, Integer detail)

方法说明

本方法提供对ssoTokenId的有效性验证

入口参数说明

#

参数名称(英文/中文)

数据类型

说明

1

ssoTokenId/单点登录令牌ID

字符串

单点登录令牌ID

2

appId

字符串

由portal分配一个appId给各个应用系统。

3

Detail

整型

如果设置为0，则接口只返回基本信息；设置为1则返回详细信息。返回参数中红色为基本信息

返回值说明：

l        结果以字符串形返回，不同参数间用逗号分割

l        认证不通过时，需要返回_returncode和_errmsg

l        每个返回值的形式为[名称]=[值]，如：

_returncode=0,_maxsessiontime=120

返回值参数说明

_returncode

函数执行状态

0 表示认证成功

1 表示认证失败

2 表示应用系统Id不正确

_principal

表示用户在对应系统中的认证身份

_uid

表示用户id

 

 

_maxsessiontime

登录的最大有效时间（分）

_idletime

ssotoken空闲时间（秒）

_maxidletime

表示ssotoken最大空闲时间

_timeleft

表示ssotoken有效剩余时间

_cn

表示用户全称

_sn

表示用户姓

_givename

表示用户名

_identitynumber

身份证号    (按照要求添加一个)

 

登录实现方式：

1.     从cookie中获取key为iPlanetDirectoryPro的内容，此内容为登录访问控制中心后产生的令牌

2.     调用validate的Webservice接口，传入3个参数（“令牌内容”,” CBPM_Library”,1）;

3.     判断放回内容，查询需要的身份信息；

4.     以对应的身份登录系统

 

 

测试环境说明：

在hosts文件中添加10.0.2.39   portal.cbpmc.cbpm，保证访问测试系统；

创建对应的appid，我已经在测试系统中创建了为图书系统使用的Appid，内容为CBPM_Library，此参数就是上述接口中使用 的参数2，

创建某个用户的映射关系，我已经为葛庆鸣创建了映射帐户为geqingming
